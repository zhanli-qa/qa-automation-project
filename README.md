# QA Automation Project

A full QA automation project covering both API and UI testing using Python, pytest, requests, and Playwright.

---

## Tech Stack
- Python
- Pytest
- Requests (API Testing)
- Playwright (UI Testing - in progress)

---

## Project Structure
api/
client/
schemas/
tests/

logger/

test_data/


---

##  Features
- API client abstraction
- JSON schema validation
- Positive and negative test coverage
- Logging support
- Scalable test structure
